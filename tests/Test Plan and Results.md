# Test Plan
Unofficial testing occurred throughout the development process. However to ensure the project met all requirements a test case was made for each of the project requirements. After the completion of the project it went through all these test cases by the developer. If there were any issues it would be fixed, then given to a peer that will also go through all these test cases to ensure better examination of the product. Utilizing the feedback and issues found by the peer the project would be adjusted. After it will go through its last round of going through the test cases.
